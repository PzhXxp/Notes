拦截器SKILL => 定制开发 SKILL

# 主要功能

分析需求输入，定位组组件，设计方案，实现代码

## 分析需求输入

### 用例1

```
在项目Greece中使用拦截器，拦截EventHandlerNew组件中uniqueId为3b3c78-84f605的组件的tabList属性，在当前界面为本地报警localAlarm或报警盒alarmBox时，将tabList中RemoteAlarmOut调整为第一位
```

### 用例2
```
隐藏预览页面底部的播放按钮
```

## 用例3

```
在某个页面的某个选项中新增一个选项，需要修改接口
```
## 运行方法

进行要点提炼，分析是否存在路由、模块、页面、组件名称、uniqueId、项目类型（customConfig/project）、项目名称（用于在 customConfig/project）下新建目录，是否涉及接口，接口每次，接口数据格式。
